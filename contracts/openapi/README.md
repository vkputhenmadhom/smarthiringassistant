# OpenAPI Contracts

Place generated OpenAPI specs here, one per service.

Recommended naming:

- `auth-service.openapi.yaml`
- `resume-parser-service.openapi.yaml`
- `candidate-matcher-service.openapi.yaml`
- `job-analyzer-service.openapi.yaml`
- `screening-bot-service.openapi.yaml`

These files are intended for compatibility checks and frontend integration validation.
